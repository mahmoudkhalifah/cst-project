<h2>features</h2>
<ol>
<li>add a task to a day</li>
<li>add a category</li>
<li>filter tasks by days</li>
<li>filter day tasks by categories</li>
<li>mark task as done</li>
<li>delete a task</li>
<li>delete all tasks of a day</li>
<li>delete all tasks and categories</li>
</ol>
